  • 실시간 메시지 송수신
  • 토픽 기반 메시지 수신
  • REST API 테스트

🔗 다중 세션 테스트

여러 STOMP WebSocket 세션을 동시에 관리하고 브로드캐스트를 테스트합니다.

  • 다중 STOMP 세션 동시 관리
  • 세션별 개별 메시지 전송
  • 토픽 기반 브로드캐스트 메시지
  • 실시간 세션 통계
테스트 시작

🎯 다중 세션 브로드캐스트 테스트 방법

애플리케이션 실행: ./gradlew bootRun
다중 세션 테스트 페이지 접속
"새 세션 추가" 버튼으로 원하는 만큼 세션 생성
"모든 세션 연결" 버튼으로 모든 세션을 WebSocket 서버에 연결
브로드캐스트 메시지 입력 후 "브로드캐스트 전송" 클릭
모든 연결된 세션에서 동시에 메시지 수신 확인

💡 추가 테스트 방법

여러 브라우저 탭으로 테스트:

여러 브라우저 탭에서 단일 세션 테스트 페이지 열기
각 탭에서 WebSocket 연결
한 탭에서 브로드캐스트 메시지 전송
다른 모든 탭에서 메시지 수신 확인
